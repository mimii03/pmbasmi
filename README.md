# PMB System

Website Penerimaan Mahasiswa Baru berbasis PHP & MySQL.

## Fitur
- Registrasi user
- Login user
- Ujian online
- Hasil kelulusan
- Admin dashboard (CRUD soal & data siswa)

## Cara Menjalankan
1. Import database `pmb_system.sql`
2. Jalankan di XAMPP (htdocs/pmb)
3. Akses di browser: localhost/pmb
